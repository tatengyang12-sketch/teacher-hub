# Teacher Hub One-Stop Shop

A single-file GitHub Pages prototype for elementary teachers.

## What is included

- Weekly planner with drag-and-drop day movement
- Disruption shifter for assemblies, fire drills, testing, or schedule changes
- Cross-curricular resource finder by subject, standard, Lexile range, and theme
- Center group generator and smartboard visual timer
- Family communication message planner with behavior badges
- Supply and wishlist tracker
- Substitute teacher packet preview and print-to-PDF export
- JSON backup and restore

## Privacy warning

This prototype uses browser localStorage only. It is not a secure school database.
Do not enter real student private information, medical information, behavior records, emergency information, phone numbers, parent messages, or real student photos into a public GitHub Pages demo.

For real school use, connect this design to a school-approved platform with authentication, permissions, audit logs, data retention rules, and district approval.

## How to publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html`.
3. Go to Settings → Pages.
4. Choose your main branch and root folder.
5. Save and open the GitHub Pages link.

## How to save a sub packet as PDF

1. Open the Sub Packet tab.
2. Fill in the routines, roster, emergency procedures, and notes.
3. Click `Export Sub Packet`.
4. In the print window, choose `Save as PDF`.
